# open_news

项目集成微博热搜、央视新闻、哔哩哔哩热门视频等多个新闻资讯平台，
基于Flutter开发，支持Android、iOS。

**项目原理:**
* 基于爬虫技术，爬取各大新闻资讯平台的数据，然后通过接口返回给客户端。
* 采用纯前端技术，基于Flutter开发，支持Android、iOS。
* 采用Flutter的WebView插件，加载H5页面，实现新闻资讯的展示。

**警告 : 该项目仅供学习使用 , 若从事违法犯罪活动 , 与本人无关。**
